Tox Crawler

This project crawls the Tox network every 5 minutes or so and graphs the number of nodes over time.

Source code: https://github.com/vikstrous/tox-crawler

The graph is automatically refreshed when new data arrives using websockets. You can also use the refresh button to force a refresh. The server is configured to collect 14000 data points before it starts discarding data. This is about a megabyte per refresh. Keeping this page open will use an average of 4kbps when the server has the maximum amount of data it can hold.
